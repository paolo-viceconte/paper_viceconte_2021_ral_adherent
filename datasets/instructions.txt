This folder is intended to contain the datasets available at https://huggingface.co/datasets/ami-iit/dnn-mpc-datasets.


In particular, the datasets are organized in the following folders:

- mocap: motion capture data collected from human motion

- retargeted_mocap: motion capture data retargeted on the robot

- IO_features: input and output features extracted from the retargeted mocap data to train the trajectory generator

- training_mirrored_D2_D3_20230910-130015: training data



Please download and extract the datasets in this folder in order to use them.
